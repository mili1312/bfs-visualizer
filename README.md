# BFS Pathfinding Visualizer (Pygame)

An interactive pathfinding simulation built with **Python** and **Pygame**, implementing the **Breadth-First Search (BFS)** algorithm on a grid.

## Features
- Interactive grid creation
- Start & End node selection
- Obstacle placement
- Real-time BFS animation
- Shortest path reconstruction

## Tech Stack
- Python
- Pygame
- Queue-based BFS implementation

## How to Run

```bash
pip install -r requirements.txt
python main.py
